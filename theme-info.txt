============================================
INSTALLATION
============================================

1. Download tetsuo-theme-v2.0.zip
2. Go to WordPress Admin > Appearance > Themes
3. Click "Add New" > "Upload Theme"
4. Choose the ZIP file and click "Install Now"
5. Click "Activate"
6. Go to Appearance > Customize to configure

============================================
QUICK START
============================================

1. Install and activate the theme
2. Install recommended plugins:
   - Advanced Custom Fields (optional)
   - Contact Form 7
   - Yoast SEO
3. Import demo content (Appearance > Import Demo Data)
4. Customize colors in Customizer
5. Set up your portfolio
6. Configure menus
