##   Overview

**NEO TOKYO CREW** is a fusion website that combines cyberpunk aesthetics inspired by iconic manga (Akira, Ghost in the Shell, BLAME!) with raw street culture graffiti art.

### 🎌 TETSUO // 鉄雄
A brutalist cyberpunk WordPress portfolio theme with glitch animations and manga-inspired design.

###   CRAFFITICREW
A street art collective brand featuring GOBE style and Lorem Ipsum poetry.

##   Features

- 🔥 **Dual Theme System** - Cyberpunk meets street art
- 💾 **Download System** - WordPress theme downloads
- 🎯 **Glitch Animations** - Cyberpunk text effects
- 🎨 **SVG Graphics** - Custom vector illustrations
- 📱 **Fully Responsive** - Works on all devices
- 🔗 **Multi-Page** - 7 interconnected pages
- 🎪 **Interactive Forms** - Crew recruitment
- 🏷️ **Tag Clouds** - Street culture elements
